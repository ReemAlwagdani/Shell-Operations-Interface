Shell Operations Interface

Overview:
- A JavaFX-based graphical interface that allows users to execute Linux shell commands through a user-friendly GUI. The application supports command execution, directory navigation, process handling, and output/error management.

Features:
- Linux Command Execution
- Directory Navigation (cd)
- Command Parameter Support
- Output and Error Handling
- Process Management
- Graphical User Interface (JavaFX)

Supported Commands:
- ls
- pwd
- mkdir
- cd
- touch
- cp
- mv
- rm
- cat
- grep
- wc
- chmod
- ps
- du
- find
- curl
- zip
- unzip

Technologies Used:
- Java
- JavaFX
- ProcessBuilder
- Linux Shell Commands

Learning Outcomes:
- Operating System Concepts
- Process Management
- Command Execution
- JavaFX GUI Development
- File System Navigation
